# 關渡心四季－2023冬季篇：學生情緒管控不易掌握，第一線導師的輔導時應留意什麼？

文／本校學生諮商中心

## 理解情緒表現的本質

每個人都有情緒，情緒表現的誘發與情境有關，呈現張力、處理方式跟人生累積處理情緒的經驗有關，而情緒從誘發到顯現的整個過程是否有所管控，跟個人主觀意願有關。

### 人類情緒的特殊性

人類跟其他生物不太一樣，情緒對其他生物來說是偏重生存的用途，但人類的情緒除了原來的生物性用途外，有更多目的性的功用在。

學生的情緒展現有其功用，學生本人是否有「要自主管控情緒」這個內在歷程是需要去區辨的。

## 校園心理健康的底線

**站在校園心理健康的底線是：**

> 情緒表現與管控只要不會危害自己的安全、不會造成旁人的困擾、不違法犯忌之下，無論學生的情緒表現如何，又是否有要自主管控情緒，這都是可以處理的議題。

### 尊重但不放任

雖然某種程度下尊重學生的情緒表達方式，但需請導師可多觀察該生與其他學生的互動狀況：

- **若有其他學生因此感覺受到干擾**：導師仍需協助受干擾學生的情緒排解
- **同時也需對該生做提醒**：讓學生了解其情緒表現對他人的影響
- **評估安全風險**：如導師判斷當下情況有危及生命或校園安全之虞，應立即進行通報

## 導師輔導時的注意事項

### 一、觀察與評估

1. **情緒表現的頻率與強度**
   - 是偶發事件還是長期模式？
   - 情緒強度是否超出一般範圍？
   - 是否影響學業或人際關係？

2. **觸發因素**
   - 特定情境或人物會引發嗎？
   - 是否有明顯的壓力源？
   - 生活作息是否規律？

3. **學生的自我覺察**
   - 學生是否意識到自己的情緒狀態？
   - 是否有意願管理情緒？
   - 過去如何處理情緒？

### 二、適切的回應方式

**當學生情緒激動時：**

1. **保持冷靜**：不被學生的情緒帶著走
2. **確保安全**：優先考量所有人的安全
3. **同理但不縱容**：理解情緒，但說明界線
4. **暫時中止**：若無法有效溝通，可約定其他時間談話

**後續跟進：**

1. **個別會談**：在情緒平穩時深入了解
2. **設定期待**：清楚說明可接受的表達方式
3. **提供資源**：介紹情緒管理的方法或資源
4. **持續關注**：定期關心，觀察變化

### 三、何時需要轉介或通報

**需要轉介至諮商中心的情況：**

- 情緒困擾持續影響學業或生活
- 出現明顯的焦慮、憂鬱症狀
- 學生表達想要尋求專業協助
- 導師感到輔導上的困難或限制

**需要立即通報的情況：**

- **有自我傷害的意圖或行為**
- **有傷害他人的意圖或行為**
- **嚴重影響課堂秩序或校園安全**
- **涉及違法行為**

### 四、保護導師自己

**設定界線：**

- 明確告知可提供協助的範圍
- 不過度承諾或承擔責任
- 保持專業距離

**尋求支持：**

- 與其他導師交流經驗
- 向學務處或諮商中心諮詢
- 參加相關輔導知能研習

**自我照顧：**

- 覺察自己的情緒負荷
- 適時休息和放鬆
- 不將學生的問題攬在身上

## 情緒管理的教育意義

### 協助學生發展情緒智能

導師在面對學生情緒議題時，同時也是在教導學生：

1. **情緒識別**：認識和命名自己的情緒
2. **情緒表達**：用適當的方式表達情緒
3. **情緒調節**：學習管理和調適情緒
4. **社會覺察**：理解情緒對他人的影響

### 創造支持性的環境

- **正常化情緒**：讓學生知道有情緒是正常的
- **示範健康的情緒處理**：以身作則
- **鼓勵求助**：營造可以談論困難的氛圍
- **連結資源**：讓學生知道有哪些支持系統

## 實務案例思考

### 案例一：情緒爆發型

**情境**：學生在導師時間因小事突然情緒爆發，大聲哭泣並指責他人。

**處理原則**：
1. 先穩定現場，確保安全
2. 給予空間讓情緒緩和
3. 個別會談了解原因
4. 評估是否需要轉介

### 案例二：沉默退縮型

**情境**：學生長期情緒低落，不與人互動，課業表現下滑。

**處理原則**：
1. 主動關懷，建立信任
2. 溫和邀請談話，不強迫
3. 持續關注變化
4. 適時建議專業協助

### 案例三：情緒勒索型

**情境**：學生用情緒威脅要求特殊待遇或改變規定。

**處理原則**：
1. 同理情緒但堅守界線
2. 清楚說明規範和原因
3. 不因情緒表現而妥協
4. 必要時尋求行政支持

## 結語

面對學生的情緒議題，導師的角色是：

- **觀察者**：敏銳察覺學生的狀態
- **陪伴者**：在學生需要時提供支持
- **引導者**：協助學生發展情緒管理能力
- **連結者**：在必要時連結適當資源

記住：

1. **情緒本身不是問題**，問題是如何表達和處理
2. **尊重個別差異**，每個學生都有自己的情緒特性
3. **保持專業界線**，不過度涉入或承擔
4. **善用資源網絡**，不需要單打獨鬥

當我們以理解和界線並重的態度，既尊重學生的情緒，又維護校園的安全與秩序，就能在輔導學生的同時，也照顧好自己。
